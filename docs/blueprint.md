# **App Name**: ProfileMirror

## Core Features:

- Profile Display: Displays the user profile with stats, friend list, and other information, closely mirroring the design provided.
- Avatar Customization: Allows users to change their avatar by providing a URL for the image.
- Banner Customization: Enables users to change the profile banner by providing a URL for the image.
- Data Storage: Persists user-uploaded avatar and banner URLs. This information will be stored using Firestore.
- Stats Display: Show a list of detailed player statistics.

## Style Guidelines:

- Primary color: Deep red (#B3261E), drawing from the highlights in the image, suggestive of intensity, energy, and passion, which aligns with the nature of competitive online games.
- Background color: Dark gray (#1A1A1A) for a modern, high-tech aesthetic; the low saturation emphasizes content.
- Accent color: Soft orange (#D97925), used for interactive elements and highlights, positioned analogous to the deep red and in order to stand out in contrast to the dark background.
- Font: 'Inter' (sans-serif) for a clean, modern, and readable UI. It will be used for both headlines and body text.
- Imitate the layout from the provided screenshot.
- Replicate the existing iconography with minimalistic icons for different profile sections.
